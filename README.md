# 習慣化サポートアプリ（Flask + localStorage）

## アプリ概要

Google スプレッドシート連携を廃止し、ブラウザの `localStorage` だけで動作する習慣化サポートアプリです。  
データはブラウザ内に保存されるため、サーバーや外部サービスの設定は不要です。

## 使用技術

- **Python 3**（3.10 以降を推奨）
- **Flask**（画面配信）
- **JavaScript + localStorage**（習慣データ/チェック履歴保存）

## 主な機能

- セット画面で習慣登録（習慣名 + 頻度）
- 頻度の保存（`daily` / `weekly1` / `weekly3`）
- 今日の習慣一覧表示
- 実施チェックのトグル（今日やった/やってない）
- 習慣削除
- リロード後もデータ保持

## ユーザーごとのデータ分離

同じ URL でも、画面右上のユーザー名を切り替えることで保存先キーが変わります。  
保存キーは `habit_support_app_v1:<ユーザー名>` です。

## セットアップ

```bash
pip install -r requirements.txt
python app.py
```

ブラウザで [http://127.0.0.1:5000](http://127.0.0.1:5000) を開いて利用します。

## 備考

- 習慣データは `localStorage` に保存されます（ブラウザ/端末ごとに独立）。
- ブラウザのストレージ削除を行うとデータは消えます。
